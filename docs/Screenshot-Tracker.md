## Presentation Tips

- Use full-screen SAP screenshots where values are clearly visible.
- Highlight document numbers with a simple rectangle if needed.
- Keep all screenshots aligned and similarly sized in the PDF.
- Add short captions below each screenshot.
- Avoid blurry mobile photos of the system screen.
